1.What is a Responsive Design?-
            Responsive Web design is the approach that suggest that design and development should respond to the users behavior and environment based on screen size. plateform and orientation.

      *The practice consists of a mix of flexible grids and layouts , images and an intelligent use of CSS media queries.

2.What is a Box Model?-
          In CSS .term "Box Model" is used when talking about design and layout.The CSS box model is essential a box that wraps around every HTML element. It consists of:Margin,Borders,Padding,and the actual content.
